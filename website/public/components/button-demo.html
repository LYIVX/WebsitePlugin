Universal Button Component

A standardized button component for consistent UI across the website.

HTML Usage

Use these classes with regular HTML elements:

Button Types

<button class="universal-btn primary">Primary Button</button>
<button class="universal-btn secondary">Secondary Button</button>
<button class="universal-btn danger">Danger Button</button>

Button Sizes

<button class="universal-btn primary small">Small Button</button>
<button class="universal-btn primary medium">Medium Button</button>
<button class="universal-btn primary large">Large Button</button>

With Icons

<button class="universal-btn primary"><i class="fas fa-edit"></i> Edit</button>
<button class="universal-btn secondary"><i class="fas fa-save"></i> Save</button>
<button class="universal-btn danger"><i class="fas fa-trash"></i> Delete</button>

Disabled State

<button class="universal-btn primary" disabled>Disabled</button>
<button class="universal-btn secondary" disabled>Disabled</button>
<button class="universal-btn danger" disabled>Disabled</button>

Link Buttons

Link Button Link Button Link Button
<a href="#" class="universal-btn primary">Link Button</a>
<a href="#" class="universal-btn secondary">Link Button</a>
<a href="#" class="universal-btn danger">Link Button</a>

Full Width

<button class="universal-btn primary full-width">Full Width Button</button>

Web Component Usage

Use the custom <universal-button> element for more functionality:

Button Types

<universal-button text="Secondary Button" type="secondary"></universal-button>
<universal-button text="Danger Button" type="danger"></universal-button>

Button Sizes

<universal-button text="Small Button" size="small"></universal-button>
<universal-button text="Medium Button" size="medium"></universal-button>
<universal-button text="Large Button" size="large"></universal-button>

With Icons

<universal-button text="Save" type="secondary" icon="fas fa-save"></universal-button>

<universal-button text="Disabled" type="secondary" disabled></universal-button>

JavaScript Usage

Create buttons programmatically:

Dynamic Buttons

// Create a primary button with an icon const primaryBtn = UniversalButtonUtil.createButton({ text: 'Dynamic Button', type: 'primary', icon: 'fas fa-star', onClick: () => alert('Button clicked!') }); // Create a link button const linkBtn = UniversalButtonUtil.createLinkButton({ text: 'Link Button', href: '#', type: 'secondary', icon: 'fas fa-link' }); // Append to container document.getElementById('dynamicButtons').appendChild(primaryBtn); document.getElementById('dynamicButtons').appendChild(linkBtn);

Update Existing Button

// Get the target button const targetButton = document.getElementById('targetButton'); // Button to trigger the update document.getElementById('updateButton').addEventListener('click', () => { UniversalButtonUtil.updateButton(targetButton, { text: 'Updated Button', type: 'danger', icon: 'fas fa-check', onClick: () => alert('Updated button clicked!') }); });
